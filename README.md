# IoTSharp.Gateway.Modbus 

IoTSharp的Modbus主站网关可以通过页面设置和添加采集映射信息可以完成自动化采集。主要功能如下:

1. 管理被采集的从机基本信息（已实现）
2. 管理采集点和IoTSharp的映射 （已实现）
3. 接收IoTSharp的下行数据和控制 (未实现)
